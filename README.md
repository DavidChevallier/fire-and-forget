## Config

> ~/.david/config.yml

```yml
apiService: openai
openai:
  endpoint: https://api.openai.com/v1/chat/completions
  model: gpt-4
olama:
  endpoint: https://api.olama.com/v1/completions
  model: lama-3
```

### models

- [openai](https://platform.openai.com/docs/models)
- [olama](https://ollama.com)
